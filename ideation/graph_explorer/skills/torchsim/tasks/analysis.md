# Analysis methods

Use `templates/analyze_trajectory.py` as the default starting point.

Always compute when available: potential/kinetic/total energy, temperature, pressure/stress, volume/cell parameters, max force, final structure.

Structural analysis: RDF, coordination, bond/angle/dihedral distributions, structure factor, local order, CNA/PTM, defect tracking.

Dynamical analysis: MSD/diffusion, VACF, vibrational density of states, residence times, hopping statistics, heat-flux autocorrelation if reported.

Mechanical analysis: elastic tensor, stress-strain, toughness proxy, crack-tip/damage metrics, local strain fields.

Interfaces/surfaces: layer density, mixing width, reconstruction, adhesion/work of separation, local stress.

Polymers/biomaterials: radius of gyration, end-to-end distance, orientation, contact maps, H-bond proxies, fiber/strand alignment.

Statistical discipline: discard equilibration, block-average, report uncertainty, verify linear regimes for transport.
